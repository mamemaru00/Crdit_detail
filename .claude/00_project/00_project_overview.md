# イオンカード明細取込システム - プロジェクト概要

## プロジェクト名
イオンカード明細取込システム（カテゴリ自動振り分け版）

## 目的
イオンカード利用明細CSVファイルを取り込み、店舗名からカテゴリを自動判定してGoogleスプレッドシートの年別シート・月別行・カテゴリ別列に金額を自動加算することで、手動入力の手間を削減し、家計簿管理を効率化する。

## 背景
現状、イオンカードの利用明細を手動でスプレッドシートに入力しており、入力作業に時間がかかっている。CSVファイルから自動的に取り込み、店舗名からカテゴリを推測して適切な列に振り分けることで、作業時間の短縮とヒューマンエラーの防止を実現する。

## 想定ユーザー
家計簿入力者（個人ユーザー）

## 主要機能
1. **CSVファイル取込** - Shift_JISエンコーディングの利用明細CSVをアップロード・解析
2. **カテゴリ自動判定** - 店舗名から外食費、日用品費、交通費などのカテゴリを自動振り分け
3. **スプレッドシート自動更新** - Googleスプレッドシートの該当する年・月・カテゴリに金額を自動加算
4. **マッピング管理** - 店舗名とカテゴリの対応関係を管理・編集

## 技術スタック
- **フレームワーク**: Flask 3.1.2（Python 3.14.0）
- **実行環境**: Docker Desktop 4.44以上
- **認証方式**: Googleサービスアカウント認証（完全自動化、ブラウザ認証不要）
- **データ管理**: JSON形式のマッピングテーブル

## 成果物
- Dockerコンテナ化されたWebアプリケーション
- ローカル環境（http://localhost:5000）で動作
- 月別・カテゴリ別の処理結果レポート
- 未登録店舗の管理機能
